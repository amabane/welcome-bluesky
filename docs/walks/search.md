# 知り合いの見つけ方

## 検索から見つける

Bluesky にはユーザー検索の方法が存在するので、知り合いのユーザー名などがはっきりしている場合は、検索から探すのが一番です。Bluesky の検索は、PC だと画面右上部の入力欄、スマホアプリだと、メニューから `Search` から行うことができます。 Bluesky の検索にはちょっと癖があるので、それについて解説します。

WIP
